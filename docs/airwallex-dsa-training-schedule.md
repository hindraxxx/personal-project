# Airwallex DSA Training Schedule

## How To Train

1. Pick one problem from the daily mix.
2. Spend 5 minutes restating inputs, outputs, constraints, and edge cases.
3. Code for 25-35 minutes.
4. Run tests or dry-run at least 3 cases.
5. Write the pattern, bug you hit, and final complexity.
