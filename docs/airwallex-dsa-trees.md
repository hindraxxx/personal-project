# Trees

## 1. Binary Tree Level Order Traversal

Given a binary tree, return values grouped by depth.

Expected pattern: queue BFS.

## 2. Validate Binary Search Tree

Given a binary tree, return whether it is a valid BST.

Practice:
- Duplicates rule.
- Use long min/max bounds to avoid integer edge bugs.

Expected pattern: DFS with bounds.

## 3. Lowest Common Ancestor

Given a binary tree and two nodes, return their lowest common ancestor.

Follow-ups:
- BST-specific version.
- Return null if one node is missing.

Expected pattern: recursive DFS.

## 4. Serialize And Deserialize Binary Tree

Design methods to convert a tree to a string and back.

Practice:
- Include null markers.
- Pick preorder or level order.
- Explain time and space.

Expected pattern: traversal with null sentinels.
